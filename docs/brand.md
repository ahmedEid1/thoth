# Thoth — Brand Guidelines

## Name & meaning

**Thoth** is the Greek transliteration of the ancient Egyptian *Ḏḥwty*, the ibis-headed god of writing, wisdom, knowledge, and scribes. The hieroglyph for "scribe" was literally an ibis — a direct symbolic match to a tool that drafts and verifies systematic literature reviews. The name itself means "[he] is like the ibis."

## Tagline

- **Primary:** Agentic systematic literature reviews with verifiable citations.
- **Secondary (brand context only):** Named for Thoth, ancient Egypt's ibis-headed god of writing, wisdom, and scribes.

## Color palette

| Role     | Name              | Hex       | Use                                                      |
| -------- | ----------------- | --------- | -------------------------------------------------------- |
| Primary  | Thoth blue (lapis lazuli) | `#1E3A8A` | Logo on light backgrounds, wordmark, primary brand accent |
| Accent   | Papyrus gold      | `#C9A961` | Secondary accent only — sparingly                        |
| On dark  | White / gold      | `#FFFFFF` / `#C9A961` | Logo when placed on dark backgrounds              |

Egyptian symbolism is intentionally light: lapis lazuli for sky/wisdom, papyrus gold for the scribe's medium. The app's existing Tailwind theme is unchanged — these colors apply to the logo and brand surfaces only.

## Logo files

| File                          | Use case                                                  |
| ----------------------------- | --------------------------------------------------------- |
| `app/icon.svg`                | Browser favicon (Next.js automatic convention), 24×24 viewBox |
| `public/thoth-logo.svg`       | Full-resolution mark for README embeds, social cards, larger displays |
| `public/thoth-wordmark.svg`   | Name + small ibis mark — used in the README header        |

All three are single-color (Thoth blue), self-contained, no external references.

## DO

- Pair the name "Thoth" with a clean, modern sans-serif.
- Use a single ibis silhouette per surface; let it breathe.
- Keep the primary blue dominant; gold is a quiet accent.
- Reference the etymology once in long-form contexts (README intro, about page).

## DO NOT

- Add papyrus textures, sandstone backgrounds, or parchment effects.
- Use hieroglyphic dividers, cartouches, or ankh icons as decoration.
- Add costume-party imagery (pyramids, sphinxes, pharaonic regalia).
- Lean on mythological metaphors in product copy or section headings — the brand carries via name and logo, not narrative.
- Apply gradients, drop shadows, or photoreal effects to the logo.
